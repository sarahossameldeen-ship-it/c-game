# c-game
This project is a 2D game developed using C++, focusing on implementing core game development concepts such as visual output handling, object interaction, and real-time game mechanics. The main goal of this project was to build a smooth and efficient game while applying fundamental programming and problem-solving skills.

## Overview
The player controls Tweety through a dynamic environment filled with obstacles and enemies.  
The objective is to survive, avoid hazards, and defeat enemies using a shooting mechanism.
The game demonstrates efficient logic handling, smooth gameplay, and interactive systems.

## Features
- Playable character (Tweety) with movement and shooting ability 
- Combat system:
  - Player can shoot and eliminate enemies
  - Enemies can damage or eliminate the player
- 4 different enemy types with unique behaviors
  (Flying Ghost - Plane shoot rockets - dragon have laser - guard of the winning gate)
- Real-time gameplay and interactions
- Environmental elements:
  - Ladder (vertical movement)
  - Elevator (moving platforms)
  - Lasers (deadly obstacles)
- Collision detection system

## Game Mechanics
- Player controls:
  - Move with (<- a , d -> , S ↓ , W ↑)
  - Climb ladders
  - Get Gun by (f), shoot bullets with (g) and drop the gun by (c)
- Enemy behaviors:
  - Enemies feature varied movement patterns and different ways to attack or eliminate the player. 
  - Interaction with player (damage on contact)
- Combat system:
  - Bullet/enemy collision detection
  - Player/enemy interaction logic
- Game-over conditions:
  - Player hit by enemies or lasers
 
 ## Technologies Used
- C++
